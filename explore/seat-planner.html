iSeat Plan

Cordova Application | HTML5 Animation

Project Overview

This mobile app was built using Apache Cordova and compiled through PhoneGap Build. The client wanted a classroom seat plan simulation with run-time update for sorting types like Alphabetically, By Gender etc.

The client also specified that the teacher can easily provide attendance and recitation on respective class sessions right away on the simulation.

  • Project owner: Dan Mark Bongar & co.
  • Skills applied: json, jquery, framework7, phaserjs, javascript, css3, html5, phonegap, apache-cordova
  • Completion date: May 24, 2016
  • Accessibility: Prototype Copy

I was responsible for the full-stack hybrid mobile application development and compilation. This is a thesis requirement of Dan Mark Bongar & co.
