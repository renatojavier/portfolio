Mobile Coloring Book

Cordova Application | HTML5 Animation

Project Overview

This mobile app was built using the Cordova Apache technology. Compiled through PhoneGap Build. The client designed the app for children above 3 years of age. They provide image patterns as rasterized image formats.

Later on the code was transformed into equivalent interactive SVG patterns. It has four (4) pattern clusters with 5 images. The objective was to completely color the segments of the pattern correctly if possible

  • Project owner: Renato Cailing & co.
  • Skills applied: json, jquery, javascript, css3, html5, phonegap, apache-cordova
  • Completion date: April 01, 2016
  • Accessibility: Prototype Copy

I was responsible for this full-stack hybrid mobile application development. This is a thesis requirement of Mr. Renato Cailing & co.
