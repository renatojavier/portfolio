Project Overview

This project is about the client's portfolio. He designed the user interface graphically, then rendered to PNG format. I was strictly required to follow the mockup pixel-perfect.

Aside from converting the design into HTML code and format, the project has applications needed to implement and I did successfully: desktop and mobile base layouts need to be scaled and rendered accordingly; scroll anchor; & bitcoin animation.

  • Project owner: Dan Martin
  • Skills applied: jquery, jquery-ui, greensock, javascript, css3, html5, html5-animation
  • Completion date: March 2017
  • Accessibility: Prototype Copy

I designed the frontend and linked applications according to his specifications.
