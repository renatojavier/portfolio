Project Overview

Special card-type animation like business card and compliment slip. Accomplished using web stack skills plus GreenSock GSAP framework (non-canvas).

I made to show each faces of the card and I added playback controls for the the animation playback that can switch the category of one-folded products.

  • Skills applied: json, jquery, javascript, greensock-gsap, css3, html5
  • Type of copy: Prototype

This was a project from UK Dev Plus, but this copy is a prototype which I have all the right to upload on my server and disclose further.
